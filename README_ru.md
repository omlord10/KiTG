# КиТГ — LaTeX-конспект по Комбинаторике и Теории Графов

LaTeX-конспект лекций по **Комбинаторике и Теории Графов** (КиТГ), лектор — Казакевич В.Г.

[English](README.md) | [Русский](README_ru.md)

## Что внутри

- **110-страничный main.pdf** с цветными боксами для определений, теорем, алгоритмов и задач
- **Пошаговые визуализации** для 12 графовых алгоритмов (DFS, BFS, Дейкстра, Беллман-Форд, Прим, Краскал, Топ. сортировка, Форд-Фалкерсон, Косарайю, Мосты, Паросочетание, Прюфер)
- **Отдельные PDF** для определений, теорем и алгоритмов (шпаргалки для печати)
- **Python-движок** для рендера `.graph` файлов в PDF через [programforyou](https://graph.online)

## Структура

```
├── build.sh              # Скрипт сборки
├── main.tex              # Основной документ
├── template.tex          # Стили и окружения
├── titlepage.tex         # Титульная страница
├── conspec01/..14/       # Содержание лекций (LaTeX)
├── python/               # Движок визуализации графов
│   ├── graph_engine.py   # Алгоритмы + рендер .graph → PDF
│   ├── generate_steps.py # Генерация пошаговых серий
│   ├── generate_all.py   # Пакетная генерация
│   ├── pfy_render.py     # Офлайн-рендер programforyou
│   └── lecNN/<algo>/     # Сгенерированные картинки шагов (.graph + .pdf)
├── figures/              # Одиночные фигуры графов
├── standalone_defs.pdf   # Все определения
├── standalone_thms.pdf   # Все теоремы
└── standalone_algos.pdf  # Все алгоритмы
```

## Зависимости

| Инструмент | Назначение | Установка (Ubuntu/Kubuntu) |
|------------|-----------|----------------------------|
| pdflatex | Компилятор LaTeX | `sudo apt install texlive-full` |
| Python 3.10+ | Движок графов | предустановлен |
| networkx | Графовые алгоритмы | `sudo apt install python3-networkx` |
| cairosvg | Конвертация SVG → PDF | `sudo apt install python3-cairosvg` |
| Node.js | Движок programforyou | `sudo apt install nodejs` |

Или установить Python-зависимости разом:
```bash
pip install -r python/requirements.txt --break-system-packages
```

## Сборка

```bash
./build.sh          # Полная сборка (картинки + PDF)
./build.sh --quick  # Только LaTeX (без генерации картинок)
```

**Результат:** `main.pdf`, `standalone_defs.pdf`, `standalone_thms.pdf`, `standalone_algos.pdf`
